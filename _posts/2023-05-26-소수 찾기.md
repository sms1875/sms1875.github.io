### 문제 링크

[문제 링크](https://www.acmicpc.net/problem/1978)

### 성능 요약

메모리: 2020 KB, 시간: 0 ms

### 문제 설명

<p>주어진 수 N개 중에서 소수가 몇 개인지 찾아서 출력하는 프로그램을 작성하시오.</p>

### 입력

 <p>첫 줄에 수의 개수 N이 주어진다. N은 100이하이다. 다음으로 N개의 수가 주어지는데 수는 1,000 이하의 자연수이다.</p>

### 출력

 <p>주어진 수들 중 소수의 개수를 출력한다.</p>

### 코드

```cc
#include <bits/stdc++.h>
using namespace std;

int decimal(int num){
    if(num < 2) return 0;
    for(int i=2; i*i <= num; i++){ 
        if(num % i == 0) return 0;
    }
    return 1;
}

int main(){
    int n;
    int count=0;
    int input;
    cin >> n;
    while(n--){
        cin >> input;
        count += decimal(input);
    }
    cout << count ;
}
```
